# Support

For support, installation questions, privacy requests, or issue reports, use the repository issue tracker:

https://github.com/misaqyrn2019/TeamHealthApps/issues

Please include:

- Jira product type
- App version or installation context
- A short description of the issue
- Steps to reproduce the issue, when applicable
- Screenshots with sensitive information removed

Do not include passwords, API tokens, private questionnaire answers, or sensitive organizational data in public issues.
